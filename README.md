🩺 Multimodal Disease Prediction System

📌 Introduction

✨ Thrilled to present our Machine Learning Project — "Health Predictor: AI-Powered Health Insights"

In today's fast-paced world, proactive healthcare is essential. Our project bridges the gap between AI and healthcare by enabling users to monitor and predict their health conditions using multiple data inputs such as facial images, voice signals, lifestyle, and medical parameters. The system provides real-time insights, early disease prediction, and personalized health recommendations through an intelligent web dashboard.

---

🎯 Problem Statement

Many people lack access to continuous and intelligent health monitoring systems. Traditional healthcare systems are reactive and focus on treatment after illness rather than prevention.

There is a need for a system that can:

- Collect diverse health data
- Predict diseases using AI models
- Provide real-time insights and recommendations

---

🎯 Objectives

- Develop a user-friendly health monitoring dashboard
- Integrate AI/ML models for disease prediction
- Analyze facial, voice, and lifestyle data
- Provide real-time analytics and insights
- Ensure secure and reliable data handling

---

⚙️ Key Highlights

- 📊 Dataset: Open-source medical & lifestyle datasets (Heart Disease, Diabetes, Skin Health)
- 🧠 Models: Random Forest, Logistic Regression, CNN (for skin analysis)
- 🎯 Accuracy: ~88% on validation data
- 🚀 Features: Facial scan, voice-based emotion detection, lifestyle-based predictions

---

🚀 Features

- 🧠 AI-based disease prediction
- 📸 Facial analysis for skin health
- 🎤 Voice analysis for emotion/stress detection
- 📊 Health parameter analysis (BMI, glucose, heart rate, etc.)
- 📈 Real-time dashboard visualization
- 🧾 Personalized health recommendations

⚙️ Installation

1. Clone the repository

git clone https://github.com/KajalHonde/Multimodal-disease-prediction-System.git

2. Navigate to the project folder

cd multimodal-disease-prediction

3. Install dependencies

Frontend: cd client
npm install

Backend: cd ../server
npm install

---

▶️ Usage

Start Backend: cd server
npm start

Start Frontend: cd client
npm start

Open in browser: http://localhost:3000

Steps

- Upload face image / provide voice input
- Enter health details
- View predictions and insights on the dashboard

---

⚙️ System Workflow

1. User inputs health data and uploads face/voice inputs.
2. Data preprocessing (image, audio, numerical data).
3. AI models analyze the inputs.
4. Disease risk prediction (Low / Moderate / High).
5. Dashboard displays insights and personalized recommendations.
